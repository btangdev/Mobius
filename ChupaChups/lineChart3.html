Option

Usage

Default

$(target).line( {option} );

options

X축 데이터 라벨

X축 데이터

{ 사원, 주임, 대리, 과장, 부장, 차장 }

String

Y축 데이터

Y축 데이터

{ 1000, 2000, 3000, 4000, 5000, 6000 }

Number

svgWidth, svgHeight

svg 넓이, svg 높이

{ svgWidth: 800, svgHeight: 360 }

Number

svgBackgroundColor

svg 배경컬러

{ svgBackgroundColor: "#fff" }

String

dataLabelDiv

데이터라벨 포지션의 기준점. 해당 값을 기준으로 포인터 위, 아래로 라벨값을 출력함.

{ dataLabelDiv: 4000 }

Number

dataLabel_up, dataLabel_down

기준점을 중심으로 값이 작으면 dataLabel_up을 사용(라벨을 위로 표시). 값이 크면 dataLabel_down사용(라벨을 아래로 표시)

{ dataLabel_up: 25, dataLabel_down: -5 }

Number

label

데이터라벨 show, hide

{ label:true }

Boolean

pointFillColor

포인터 내부 컬러

{ pointFillColor:"#fff" }

String

pointStrokeColor

포인터 보더 컬러

{ pointStrokeColor:"dodgerblue" }

String

pointRad

포인터 반지름

{ pointRad:4 }

Number

pointExpand

마우스 오버 이벤트에 포인터 확장 유무

{ pointExpand:false }

Boolean

pointExpandRate

포인터가 확장되는 비율. 마우스 오버시 기본값의 1.75배율로 확장

{ pointExpandRate:1.75 }

Number

colorPattern

복합적인 그래프의 경우 각각의 그래프 컬러 패턴을 배열로 저장.

{ colorPattern:["dodgerblue"] }

String

그래프 내부의 x,y좌표 그리드

{ x_grid:true, y_grid:true }

dasharrayX, dasharrayY

그래프 내부 그리드 스타일. 도트로 출력되며, 도트의 길이와 간격으로 조절함.

{ dasharrayX:"2 2", dasharrayY:"2 0" }

Number

gridLineColor

{ gridLineColor:"#eee" }

String

legend

범례 show, hide

{ legend:false }

Boolean

regions_start, regions_end

그래프의 특정구간을 점선으로 구분함. 컬러변경(X), 라인스타일변경(X)

{ regions_start:1, regions_end:3, regions_start_2:4, regions_end_2:5 }

Number

각 구간에 마우스 오버시 X축 데이터를 중심으로 세로 라인을 출력

{ focusLine:false }

{ axis_x_outer:true, axis_y_outer:true }

dataLabelColor, dataLabelFontSize

데이터 라벨의 컬러와 폰트 사이즈 조정

{ dataLabelColor:"deepblue", dataLabelFontSize:"12px" }

Number, String

axis_y_font_size, axis_y_font_color

Y축 라벨의 컬러와 사이즈

Number, String

{ axis_x_positionY: 9, axis_y_positionY: 1 }

Number

X, Y축 라벨의 tick 라인의 컬러. 따로 삭제할 수 없으니 배경컬러와 동일하게 조절하면 보이지 않음.

{ axis_x_tick_line:"#000", axis_y_tick_line:"#ddd" }

String

labelPosition

기준값을 중심으로 포인터의 상 하로 레이블 출력유무

if ( option.labelPosition ){
    for ( i; i < $yAxisLabel.length; i++ ){
        d3.selectAll('.c3-texts-' + $yAxisLabel[i] + ' .c3-text').each(function(d) {
            if ( d === undefined ) return;
            d.value > option.dataLabelDiv ? getDataLabelDiv( d, option.dataLabel_up, i ) : getDataLabelDiv( d, option.dataLabel_down, i );
        });
    }
}

Boolean

format

C3에서 제공하는 포맷형태가 있지만, Label의 Boolean과 독립적인 옵션으로 제공되지 않기 때문에 새로 작성함.

function getDataLabelVal( d, i ){
    if ( d.value === 0 ) {
        $($thisTarget + ' .c3-texts-' + $yAxisLabel[i] + '.c3-texts .c3-text-' + d.index).text('');
    }else{
        var t =$($thisTarget + ' .c3-texts-' + $yAxisLabel[i] + '.c3-texts .c3-text-' + d.index).text();
        $($thisTarget + ' .c3-texts-' + $yAxisLabel[i] + '.c3-texts .c3-text-' + d.index).text(t + option.format);
    }
}

String

zeroValue

데이터가 0일때 데이터레이블 출력 유무

if ( option.zeroValue ){
    for ( i; i < $yAxisLabel.length; i++ ){
        d3.selectAll($thisTarget + ' .c3-texts-' + $yAxisLabel[i] + ' .c3-text').each(function(d) {
            getDataLabelVal(d, i);
        });
    }
}

Boolean

rectView

그래프 배경의 컬러를 홀수, 또는 짝수에 따라 구분컬러를 넣어줄지 선택.

if ( option.rectView ){
    $($thisTarget).find('.c3-event-rects-single rect').each(function(i){
        var cls = $(this).attr('class').split(' ');
        var clsNum = cls[cls.length-1];
        $($thisTarget).find('.c3-event-rects-single rect:odd').attr('fill',option.rectOddColor);
        $($thisTarget).find('.c3-event-rects-single rect:even').attr('fill',option.rectEvenColor);
    });
}else{
    $($thisTarget).find('.c3-event-rects-single rect:odd').attr('fill', 'transparent');
    $($thisTarget).find('.c3-event-rects-single rect:even').attr('fill', 'transparent');
}

Boolean

rectOddColor

그래프 배경의 컬러를 홀수에 넣어줌.

$($thisTarget).find('.c3-event-rects-single rect:odd').attr('fill',option.rectOddColor);

String

rectEvenColor

그래프 배경의 컬러를 짝수에 넣어줌.

$($thisTarget).find('.c3-event-rects-single rect:even').attr('fill',option.rectEvenColor);

String

type

그래프 타입

{ type: option.type }
